Executive Summary

The Water Resource Management System is a comprehensive web-based application designed to monitor, analyze, and manage water quality parameters in real-time. This system focuses on two critical water quality indicators: pH levels and Heavy Metal concentrations, providing essential data for environmental monitoring and public health protection.

Key Achievements:

  • ✅ Real-time water quality monitoring
  • ✅ Cloud-based data storage with Firebase
  • ✅ Interactive analytics dashboard
  • ✅ USB sensor integration capability
  • ✅ Secure admin controls
  • ✅ Responsive web design
  • ✅ GitHub Pages deployment

Parameter Specifications

pH Level Monitoring

  • Range: 6.5 - 8.5 (Normal)
  • Unit: pH scale (0-14)
  • Validation: Real-time range checking
  • Visualization: Gauge and trend charts
  • Importance: Critical for water safety and chemical reactions

Heavy Metal Detection

  • Range: 0.01 - 0.5 mg/L (Safe)
  • Unit: Milligrams per Liter
  • Validation: Toxicity level alerts
  • Visualization: Progress bars and analytics
  • Importance: Toxic to human health, environmental indicator

Database Design

Firebase Firestore Schema

// Reading Document Structure { id: timestamp, timestamp: "2025-01-22T10:30:00.000Z", date: "1/22/2025", time: "10:30:00 AM", location: "Sample Location", ph: 7.2, heavyMetal: 0.15, source: "manual" | "sensor" }

Data Validation Rules

  • pH Range: 0.0 - 14.0 (Warning outside 6.5-8.5)
  • Heavy Metal: 0.001 - 10.0 mg/L (Warning above 0.5)
  • Location: Required string field
  • Timestamp: Automatic ISO format
  • Source: Manual entry or sensor detection
