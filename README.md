# Water Resource Management System

A web-based application for monitoring and managing water quality parameters including pH levels and heavy metal contamination.

## Features

- Real-time water quality monitoring
- pH and lead level tracking
- Data visualization with charts
- Local storage with Firebase backup
- Responsive web interface

## Files Structure

### Core Application Files
- `index.html` - Main application interface
- `app.js` - Core application logic
- `ui.js` - User interface management
- `style.css` - Application styling

### Configuration & Setup
- `firebase-config.js` - Firebase database configuration
- `config.js` - Application configuration
- `jsconfig.json` - JavaScript project configuration

### Additional Features
- `admin.js` - Administrative functions
- `charts.js` - Data visualization
- `simple-sensor.js` - Sensor simulation

### Deployment
- `.github/workflows/pages.yml` - GitHub Pages deployment
- `.nojekyll` - GitHub Pages configuration

## Usage

1. Open `index.html` in a web browser
2. Add water quality readings through the interface
3. View data analysis and trends
4. Export or manage data as needed

## Technologies

- HTML5, CSS3, JavaScript
- Firebase Firestore (optional)
- Chart.js for data visualization
- Bootstrap for responsive design
